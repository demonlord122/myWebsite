Εχω προσθέσει και λίγο javascript έτσι για πλάκα, αν θέλετε τρέξτε το accesspage.html το οποίο θα σας πετάξει στο main.html που ειναι η κεντρική σελίδα μετά από κάτι, τώρα αν δεν θέλετε άπλα ανοίξτε το main.html. Επισεις θα πρεπει να χρησημοποιησετε καποιον compiler που να μπορεί να τα τρεξει ολα μαζί, γιατι πρεπει να τα εχετε ολα ανοικτα για να δίτε ολα αυτα τα html files, ειναι συνδεδεμένα με navigation bar.
